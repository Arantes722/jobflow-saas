# 🚀 Applytic

Applytic é uma aplicação web para gestão de candidaturas de emprego.

Permite aos utilizadores registar, acompanhar e organizar as suas candidaturas num único dashboard.

---

# ⚙️ Stack

- Next.js
- TypeScript
- TailwindCSS
- Supabase (Auth + Database)

---

# ✨ Funcionalidades

- Autenticação de utilizadores (registo e login)
- Dashboard privado por utilizador
- Criação de candidaturas de emprego
- Listagem de candidaturas
- Dados separados por utilizador (segurança com RLS)

---

# 🧱 Estrutura

- Frontend: Next.js (App Router)
- Backend: Supabase
- Base de dados: PostgreSQL (via Supabase)

---

# 💡 Objetivo

Projeto desenvolvido com o objetivo de aprender desenvolvimento full-stack moderno e construção de SaaS reais.
